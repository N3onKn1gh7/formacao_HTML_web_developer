De onde surgiram as tags do HTML?

As Tags do HTML foram inventadas por Tim Berners Lee. Ele é o criador da World Wide Web (WWW) e desenvolveu o HTML (HyperText Markup Language) no final dos anos 1980 e início dos anos 1990 enquanto trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear).Tim Berners-Lee criou as primeiras versões do HTML para permitir a formatação de textos e a criação de links entre documentos, facilitando o compartilhamento de informações entre pesquisadores. A primeira proposta formal do HTML foi publicada em 1991, e desde então, a linguagem evoluiu bastante, passando por várias revisões e atualizações, como HTML 2.0, HTML 3.2, HTML 4.01, XHTML, até chegar ao HTML5, que é a versão mais recente e amplamente utilizada hoje em dia. Então, podemos dizer que Tim Berners-Lee é o "pai" das tags HTML, pois ele estabeleceu os princípios e as primeiras especificações da linguagem que se tornaria a base da web como a conhecemos hoje.

Tags h1,h2,h3,h4,h5,h6:

Tags h são usadas para identificar o cabeçalho ou título de um texto. Onde os valores de 1 a 6 indicam a sua importância no texto. Por exemplo: h1 normalmente é usada para o titulo principal equanto h2 e h3 para subtítulos e assim por diante. Geralmente não se usa mais do que uma tag h1 em uma página

Exemplo:

<h1>Este é um cabeçalho de nivel 1</h1>
<h2>Este é um cabeçalho de nivel 2</h2>
<h3>Este é um cabeçalho de nivel 3</h3>
<h4>Este é um cabeçalho de nivel 4</h4>
<h5>Este é um cabeçalho de nivel 5</h5>
<h6>Este é um cabeçalho de nivel 6</h6>

Tag small

É usada para representar texto de menor importância ou de menor escala. Visualmente, ela geralmente exibe o texto em uma fonte menor do que o texto circundante. Tradicionalmente é utilizada para avisos legais, notas de rodapé, disclaimers ou qualquer outro texto que deva ser destacado de maneira menos proeminente.

Exemplo do código:

<p>O céu do porto tinha a cor de uma televisão sintonizada num canal fora do ar</p>
<small>"trecho retirado do livro Neuromancer de Willian Gibson"</small>

Como fica na pagina:

O céu do porto tinha a cor de uma televisão sintonizada num canal fora do ar

"trecho retirado do livro Neuromancer de Willian Gibson"

Tag a

Usada para criar hyperlinks, que são links clicáveis que permitem aos usuários navegar entre diferentes páginas da web, partes da mesma página ou mesmo abrir recursos externos, como arquivos ou endereços de e-mail. É uma das tags mais importantes e amplamente usadas em HTML.

Exemplos:

<a href="https://wwww.paginaweb.com">visite a página web</a>
<a href="mailto:email@exemplo.com">envie um e-mail</a>
<a href="documentos/arquivo.pdf">baixe o arquivo PDF</a>
<a href="https://www.exemplo.com" target="_blank">abrir em uma nova aba</a>

Como link para uma seção:

<a href="#secao">Vá para a seção</a>

Nesse caso de haver um identificador de referência para o link:

<h2 id="secao">secao alvo</h2>

Tag sub

O marcador "Subscript Element" ou subscrito é utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo.

exemplo:

O benzeno é um hidrocarboneto aromático cuja fórmula química é:

<p>C<sub>6</sub>H<sub>6</sub> - Seis átomos de carbono e seis de hidrogênio.</p>

C6H6 - Seis átomos de carbono e seis de hidrogênio.

Tag blackquote

Tag font

A tag font era usada em versões antigas do HTML para aplicar estilos de fonte diretamente no HTML, como cor, tamanho e tipo de fonte. No entanto, essa tag foi descontinuada e considerada obsoleta no HTML 4.01 e removida no HTML5. Atualmente, o uso de CSS é a maneira recomendada para estilizar texto e aplicar estilos de fonte.

Embora não seja mais recomendada, aqui está a sintaxe básica da tag font:

<font color="red" size="4" face="Arial">Exemplo de utilização da tag font</font>

Tag del

A tag del em HTML é usada para marcar texto que foi deletado ou removido de um documento. Visualmente, ela geralmente renderiza o texto com uma linha riscada (strike-through), indicando que o conteúdo não é mais válido ou foi substituído. 

<del>Texto deletado</del>

Tag abbr

A tag abbr em HTML é usada para marcar abreviações ou acrônimos. Ao usar essa tag, você pode fornecer uma explicação ou expansão do termo abreviado, o que melhora a acessibilidade e a compreensão do conteúdo.

Exemplo:

Ao passar o ponteiro do mouse em cima da sigla ONU, uma pequena legenda aparece na tela indicando Organização das Nações Unidas 

<p>A <abbr title="Organização das naçoes unidas">ONU</abbr>Foi fundada em 1945.</p>

A ONU Foi fundada em 1945.
